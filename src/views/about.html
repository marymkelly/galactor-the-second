About Project - Galactor

What is this website?

This site hosts a live-example of software development project related to Mary Kelly's portfolio.

What does it do? *

Users can search and view nearby stars based on their geographical location.
    Some Notes:
  • The search results are currently limited to return 10 at a time.
  • If the site seems slow to load at first, this is likely because the server needs to get up and running. Give it a moment :)
  • If you encounter issue when trying to search through 'use my location', it is possible that location permissions need to be approved through your browser.
* A detailed writeup of the project background and implementation can be found on my portfolio site.
